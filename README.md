# CheckList-Movil

Sistema de formularios web para inspección de vehículos y equipos móviles con generación de PDF.

## Formularios Disponibles

- **CHECK-MOVIL-0001**: Vehículos
- **CHECK-MOVIL-0002**: Rodillo Compactador
- **CHECK-MOVIL-0003**: Carro de Arrastre
- **CHECK-MOVIL-0004**: Retroexcavadora

## Características

- Formularios responsivos para móvil, tablet y escritorio
- Generación de PDF con firma digital
- Diseño consistente y profesional
- Validación de campos obligatorios

## Uso

Abrir `index.html` en un navegador para acceder al menú principal de formularios.
